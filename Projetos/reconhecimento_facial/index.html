Reconhecimento Facial Usando CNN

Criei este projeto simples meramente por questões didáticas já que estava estudando CNN (Convulational Neural Networks). Basicamente eu montei um modelo do zero, não utilizei nenhuma arquitetura pronta, como RES-NET ou ALEX-NET, ao invés montei eu mesmo, de forma bem amadora mas para simplesmente entender a lógica por trás do CNN.
